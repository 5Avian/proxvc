# Contributing

Thanks for considering to contribute to this project!
For your PRs to be accepted, they should follow these guidelines:

- Your PR and commits have an appropriate description/message
- Your code follows the conventions for this project
  - Follow [Java code conventions](https://www.oracle.com/technetwork/java/codeconventions-150003.pdf)
  - Comments start with 1 space
  - Overridden methods should be annotated with `@Override`
  - Prefer referring to `this` implicitly
  - Use no more than 1 consecutive blank line
  - Ensure names are descriptive
